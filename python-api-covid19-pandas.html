Uso de la API del Covid19 con Pandas¶

En esta prctica, vamos a usar la API COVID-19, desarrollada por Kyle Redelinghuys y basada en los datos que proporciona la Universidad Johns Hopkins.

Instalación librerías¶

Para empezar, tenemos que instalar la librera necesaria para proyectar los datos que obtengamos sobre la incidencia de la COVID-19. Para ello, usamos el comando pip install pandas. Para especificar que lo que estamos haciendo es ejecutar un comando bash, ponemos un ! al inicio de la lnea.

Configuración de Pandas¶

Una vez instalada, tenemos que importarla al proyecto que estamos realizando. Para ello, usamos la funcin import, e importamos la librera como pd, de forma abreviada, tal y como sigue a continuacin.

In [2]:
import pandas as pd

Crear variable¶

Ya est instalada la librera Pandas y la hemos importado. Ahora ya podemos empezar a crear las variables necesarias para, posteriormente, realizar los grficos con los datos de la API. Lo primero ser definir la variable url, que ser la que le diga a Python el origen de los datos.

In [3]:
url = 'https://api.covid19api.com/countries'

Crear Dataframe¶

Una vez tenemos el enlace donde se encuentran los datos, podemos empezar a mostrarlos. Lo primero ser crear un dataframe, una tabla con filas y columnas en la que aparecern los datos. Para ello, usamos la funcin de Pandas que lee el formato .json, en el que estn los datos de la API: pd.read_json(url). Al poner (url), le estamos indicando a Pandas cules son los datos que tiene que leer.

In [4]:
df = pd.read_json(url)
